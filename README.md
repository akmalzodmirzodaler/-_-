# Юрист Бот

Telegram-бот для изучения законодательства Республики Таджикистан.

## Функционал
- Тесты по Конституции РТ, Трудовому кодексу, Налоговому кодексу и др.
- Q&A система на основе базы знаний
- Развертывание на Render

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте переменные окружения в `.env`
4. Запустите: `python src/main.py`

## Развертывание
- Используйте Render для хостинга
- Настройте переменные окружения в Render
